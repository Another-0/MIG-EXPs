# 有向/旋转目标检测

## 环境安装
1. 安装pytorch（1.13.1, 镜像内省略pytorch-cuda=11.6）
```bash
conda install pytorch==1.13.1 torchvision==0.14.1 torchaudio==0.13.1 pytorch-cuda=11.6 -c pytorch -c nvidia
```

2. 安装openmin
```
pip install -U openmin
```

3. 安装mmrotate 1.x
```
mim install mmrotate==1.0.0rc1
```
